Instructions

  • Default base link for Java servlet is http://localhost:8080/quiz-rest-services/rest/quiz/. But if it is not or you see "Please update base URL for REST service" error please update the base URL with correct port and extensions
  • Search Quiz searches all quizzes that has a name match with given search parameter
  • If you click a quiz on search results, it will prepare Quiz Solve section
  • To create a quiz you need to fill quiz name, user nick and have at least 1 saved question on Create Quiz section
  • To save a question you need to fill content and all 4 choices
  • If you fill a question but don't save, it will not be included in the quiz
  • You can't create a quiz with an already existing name
  • To delete a quiz on Delete Quiz section you need to give correct quiz name and nick of owner (low level protection simulation)
